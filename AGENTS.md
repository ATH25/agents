# AGENTS

## How to work in this repo
- Work from the repository root and keep changes scoped to the requested task.
- Favor clear, incremental updates over large refactors unless explicitly requested.
- Document assumptions briefly when they affect implementation decisions.

## MCP preference
- Prefer MCP server `pycharm_2025` when it is enabled (PyCharm 2025.3.3).
- Use `pycharm_2026` only when `pycharm_2025` is not available (PyCharm 2026.1 EAP).
- If `pycharm_2026` shows no tools, ignore it when running PyCharm 2025.3.3.

## Workflow
- For any non-trivial task: first search the repo for relevant files, then open the key files, then propose changes, then apply edits.
- Keep edits minimal and avoid formatting-only churn.
- Run available checks/tests after changes when reasonable.
