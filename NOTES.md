# Notes

- This repository supports a 6-week course on building and deploying autonomous AI agents.
- The curriculum covers OpenAI Agents SDK, CrewAI, LangGraph, AutoGen, and MCP.
- Week 1 focuses on agent foundations and core concepts.
- Weeks 2-5 progress through OpenAI SDK, CrewAI orchestration, LangGraph stateful workflows, and AutoGen programming.
- Week 6 focuses on MCP (Model Context Protocol) for extending agent capabilities.
- Content is organized into weekly labs (`1_foundations/` to `6_mcp/`) plus supporting `guides/`, `assets/`, and `setup/` directories.

## Next Steps

- Run the setup instructions from the `setup/` directory for your operating system.
- Start with `1_foundations/` and work through the week folders in order.
