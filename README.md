# Master AI Agentic Engineering - build autonomous AI Agents

### 6 week journey to code and deploy AI Agents with OpenAI Agents SDK, CrewAI, LangGraph, AutoGen and MCP

Welcome to the start of your 6 week adventure into the powerful, astonishing and often surreal world of Agentic AI. This project contains the materials and labs for the course.

### Course Outline

- **Week 1: Foundations** - Introduction to AI Agents and core concepts.
- **Week 2: OpenAI** - Building agents with the OpenAI SDK.
- **Week 3: CrewAI** - Multi-agent orchestration.
- **Week 4: LangGraph** - Building stateful, multi-agent applications.
- **Week 5: AutoGen** - Programming agentic workflows.
- **Week 6: MCP (Model Context Protocol)** - Extending agent capabilities.

### Getting Started

Detailed setup instructions for various operating systems can be found in the `setup/` directory.

### Project Structure

- `1_foundations/` to `6_mcp/`: Weekly course content and labs.
- `guides/`: Essential guides and notebooks.
- `assets/`: Images and other static assets.
- `setup/`: Installation and troubleshooting guides.
